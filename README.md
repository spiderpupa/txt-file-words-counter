# txt-file-words-counter
## 문자 수 세기 예제
### 소개
- Udacity NLP 과정 중 스팸메일 분석기 예제에서 나온 분석용 파일 세기
- 영문 분석에는 잘 적용되는 것을 확인
- easygui 라이브러리를 사용하여 windows 상에서 파일탐색기를 통해 파일 불러오기
- regex 라이브러리를 불러와 정규표현식을 사용
